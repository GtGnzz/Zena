Zena Network

Survival Economy

Server survival economy adalah server dengan game mode bertahan hidup open world yang mengunakan mata uang di dalam server untuk membeli barang kebutuhan survival.

Mini Games

Server Mini Games Adalah Map Open World Dan Memiliki Beberapa Menu Games Seperti Bedwars - Parkour Labirin Dan Lain Lain

Pvp

Server Pvp Adalah Tempat Untuk Bertarung Dalam Mode Survival Secara Duel Atau Squad. Gunakanlah Armor Dan Tools Yang Kuat Untuk Menahan Dari Serangan Damage Orang Lain!

Shop Ui

Shop Ui Adalah Tempat Dimana Kalian Membeli Sesuatu Tanpa Kembali Ke Lobby. Untuk Membuka Shop Ui Kalian Bisa Mengetikan /shop. Ada Beberapa Shop Lain Seperti Eshop Yaitu Untuk Membeli Item Sihir

Job Menu

Job Menu Adalah Pekerjaan Dengan Beberapa List. Selesaikan Tugasmu Dan Dapatkan Hadiah Berupa Uang Untuk Keperluan Lain. Setiap Uang Yang Didapatkan Tergantung Job Apa Yang Kalian Sedang Pekerjakan

Reward

Kamu Bisa Claim Reward Harian Untuk Mendapatkan Item Dari Reward. Reset Reward Direset Setelah 24 Jam Setelah Kamu Melakukan
